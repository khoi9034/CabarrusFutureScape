# CFS Two-Minute Demo Script

Good morning. This is Cabarrus FutureScape, an internal planning intelligence
prototype for parcel-level due diligence and constraint review.

CFS brings together parcel records, zoning context, permits, FEMA flood
constraints, public school assignment, transportation context, utility proxy
context, and model governance in one review workspace. The goal is not to
automate decisions. The goal is to help staff see what is known, what needs
review, and what caveats matter.

I will start in Overview at the CFS Command Center. It gives four compact
actions: Search Parcel, Countywide Intelligence, Save Snapshot, and Open
Snapshots. The Layers rail starts collapsed so the map stays central. Then I use the parcel search. For this demo I am using
`CFS-PARCEL-0149726579`. When the parcel is selected, the map and dashboard
focus on the same record, and the active selection overlay confirms the parcel,
owner, zoning, and data source.

Next I save a Planning Snapshot for the report. This captures the selected
parcel when available, active context, map image when the
SceneView screenshot is available, caveats, and explainable metrics so the
executive summary is based on a deliberate saved view rather than a loose screen
of numbers. Multiple snapshots can be saved in the local Planning Snapshot
Library, opened for review, or deleted when they are no longer needed.

The Planning Snapshot has an Explain the Numbers view. Each major item names
what the number means, where it came from, why it matters, what caveat applies,
and what staff should review next. The Executive Summary then turns the saved
snapshot into a report-style planning memo, and Print Executive Summary sends
that report to print.

Methodology is where CFS is intentionally transparent. The current best
internal model research variant is Zoning + Transportation + Tax/Value, but the
model is not production-ready. CFS does not show exact parcel probabilities,
parcel-level ranking classes, or public prediction endpoints.

The next data needed includes official school capacity and enrollment,
official rezoning case history, future land use, utility capacity, and dated
transportation project data.

In short, CFS is ready as a demo-safe parcel due diligence prototype: useful
for review, transparent about limitations, and careful not to overclaim.
