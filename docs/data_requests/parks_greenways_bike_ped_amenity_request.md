# Parks, Greenways, and Bike-Ped Amenity Data Request

## CFS Context

Cabarrus FutureScape is a planning intelligence prototype for parcel review,
constraint overlays, and internal model research. It is not a public prediction
decision system.

## What Data Is Requested

Requested data includes planned parks, greenways, trails, bike-ped facilities,
trailheads, open-space priority areas, and future amenity investments.

## Why It Is Useful

Future amenities can influence place-based planning context, residential
attractiveness, public access, and long-range development interpretation.

## Acceptable Formats

- REST service
- File geodatabase
- Shapefile
- GeoPackage
- CSV with geometry
- Excel with project location fields
- Generalized amenity corridors or planning polygons

## Minimum Needed Fields

- project or amenity name
- amenity type
- status
- expected year, if available
- funding status
- plan source
- jurisdiction
- notes

## Optional Helpful Fields

- project phase
- implementation priority
- adoption year
- connection type
- public access status
- responsible department
- geometry confidence

## Sensitivity Note

Unfunded concepts and draft alignments should be separated from funded or
adopted projects. Generalized geometry is acceptable for conceptual corridors.

## Prototype Note

CFS will label this as future amenity context. It will not be used as a public
prediction decision system.
