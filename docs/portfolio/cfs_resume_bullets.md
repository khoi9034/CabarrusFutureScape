# CFS Resume Bullets

## Short Project Description

Built Cabarrus FutureScape, a full-stack parcel planning intelligence prototype
that combines Next.js, FastAPI, PostGIS, ArcGIS Maps SDK, and Python pipelines to
support parcel due diligence, constraint review, executive reporting, and
responsible model governance for 110,000+ parcels.

## Three Concise Resume Bullets

- Built a full-stack GIS planning intelligence prototype using Next.js,
  TypeScript, FastAPI, PostGIS, ArcGIS Maps SDK, and Python pipelines to unify
  parcel search, due diligence, constraint review, and reporting for 110,000+
  parcels.
- Integrated parcel, zoning, permit, FEMA flood, school assignment,
  transportation, utility proxy, and model-governance data into a parcel-centric
  review workflow with Executive Print and Methodology views.
- Developed responsible model-governance documentation for internal development
  research, keeping prediction probabilities and parcel-level ranking classes
  out of the public UI.

## Five Detailed Resume Bullets

- Designed and implemented Cabarrus FutureScape, a parcel-based planning
  intelligence prototype combining React/Next.js, FastAPI, PostGIS, ArcGIS
  SceneView, and Python ETL/QA pipelines.
- Built a selected-parcel due diligence workflow that hydrates parcel geometry,
  focuses a 3D map, renders parcel cage/highlight context, and summarizes
  zoning, permits, flood, schools, transportation, and utility proxy data.
- Created spatial data and API integrations for FEMA flood constraints, FEMA
  flood zones, school attendance-zone assignments, school utilization caveats,
  development hotspots, and transportation accessibility context.
- Produced internal model research and governance artifacts over 1.4M+
  parcel-year feature rows while preserving safety flags:
  `model_active=false`, `prediction_probability_available=false`, and
  `production_ready=false`.
- Packaged the prototype for leadership and portfolio review with demo scripts,
  screenshot evidence, an Executive Print report, a capability matrix, a
  technical architecture summary, and a what-not-to-claim guide.

## Three Technical Resume Bullets

- Developed TypeScript API clients, React hooks, and dashboard components that
  connect FastAPI endpoints to parcel intelligence, flood, school,
  transportation, permit, and aggregate model-governance views.
- Built PostGIS-backed spatial workflows for parcel overlays, zoning history,
  flood constraints, school assignment, and transportation feature engineering,
  with QA outputs and validation summaries.
- Implemented ArcGIS SceneView layer interactions for selected parcel focus,
  parcel cage highlighting, flood constraints, FEMA zones, school utilization,
  and development hotspot overlays.

## Three Leadership / Communication Resume Bullets

- Translated complex planning, GIS, and model-governance outputs into
  leadership-ready materials including an executive one-pager, supervisor
  handoff memo, capability matrix, demo scripts, and data request packet.
- Established clear communication guardrails distinguishing verified data,
  presentation-derived values, proxy infrastructure context, and internal model
  research.
- Created a professional demo flow that helps planning and GIS stakeholders
  understand parcel context, constraints, model limitations, and next data
  priorities without overclaiming production readiness.
