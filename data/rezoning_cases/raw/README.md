# Rezoning Cases Raw Files

Place original official rezoning, map amendment, conditional zoning, and
planning case records here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files with parcel IDs, or generalized case
polygons.

Raw files should not be edited. Future ingestion will validate approval dates,
decision/status fields, old/new zoning fields, parcel links, and geometry
before use.
