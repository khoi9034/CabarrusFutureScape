# Transportation Projects Raw Files

Place original local transportation project, road widening, road extension,
intersection improvement, corridor, CIP, and concept network files here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files, or generalized project corridors.

Raw files should not be edited. Future ingestion will validate status, dates,
geometry, project type, and funding fields before use.
