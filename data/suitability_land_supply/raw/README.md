# Suitability and Land Supply Raw Files

Place original plan-based suitability, land supply, developability, and
constraint-weighted opportunity files here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files with parcel links, or generalized planning
polygons.

Raw files should not be edited. Future ingestion will validate methodology
fields, source dates, category mappings, geometry, and limitations before use.
