# Development Pipeline Raw Files

Place original development pipeline, subdivision approval, site plan, and active
development review files here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files with parcel IDs, or generalized project
polygons.

Raw files should not be edited. Future ingestion will validate status,
approval dates, project type, units or square footage, parcel links, and
sensitivity before use.
