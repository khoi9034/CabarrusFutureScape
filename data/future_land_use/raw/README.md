# Future Land Use Raw Files

Place original countywide future land use, small-area plan, place type, growth
area, and adopted policy geography files here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files with parcel links, or generalized planning
polygons.

Raw files should not be edited. Future ingestion will validate schema,
geometry, coordinate system, date fields, jurisdiction coverage, and sensitivity
before any data is used in CFS.
