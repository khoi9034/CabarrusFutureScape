# Parks, Greenways, and Bike-Ped Raw Files

Place original planned parks, greenways, trail, bike-ped, trailhead, and future
amenity files here.

Expected formats include REST exports, geodatabases, shapefiles, GeoPackages,
CSV files with geometry, Excel files, or generalized amenity corridors.

Raw files should not be edited. Future ingestion will validate project status,
expected year, funding status, geometry, and source plan before use.
